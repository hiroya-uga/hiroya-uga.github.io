本サイトの一部のツールは、スマートフォンやPCにインストールしてアプリのように使えます。インストールするとホーム画面やアプリランチャーから直接起動でき、全画面表示でご利用いただけます。

## インストールする

### iPhoneやiPad

| 手順 | 操作                                                                           |
| :--: | ------------------------------------------------------------------------------ |
|  1   | Safariでツールのページを開きます                                               |
|  2   | 画面下部（または上部）にある共有ボタン（四角に上矢印のアイコン）をタップします |
|  3   | 表示されるメニューをスクロールして「ホーム画面に追加」をタップします           |
|  4   | 名前を確認して右上の「追加」をタップします                                     |

### Android

ページ上部にインストールを促すバナーが自動的に表示される場合は、バナーからインストールできます。

| 手順 | 操作                                                             |
| :--: | ---------------------------------------------------------------- |
|  1   | Chromeでツールのページを開きます                                 |
|  2   | 画面右上のメニューアイコン（⋮）をタップします                    |
|  3   | 「ホーム画面に追加」または「アプリをインストール」をタップします |

### PC（Chrome・Edge）

| 手順 | 操作                                                               |
| :--: | ------------------------------------------------------------------ |
|  1   | ChromeまたはEdgeでツールのページを開きます                         |
|  2   | アドレスバーの右端に表示されるインストールアイコンをクリックします |
|  3   | 表示されるダイアログで「インストール」をクリックします             |

## アンインストールする

### iPhoneやiPad、Android

ネイティブアプリ同様、ホーム画面のアイコンを長押ししてアンインストールできます。

### PC（Chrome・Edge）

| 手順 | 操作                                                             |
| :--: | ---------------------------------------------------------------- |
|  1   | インストール済みのアプリウィンドウを開きます                     |
|  2   | タイトルバー右上のメニューアイコン（⋮ または …）をクリックします |
|  3   | 「アンインストール」をクリックします                             |

または、アドレスバーに `chrome://apps`（Chrome）または `edge://apps`（Edge）と入力してアプリ一覧を開き、該当するアプリを右クリック（またはメニューを開いて）「削除」または「アンインストール」を選択することもできます。
