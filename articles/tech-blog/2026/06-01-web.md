WEBサイト。webサイト。  
うーんもやもやする。

Webサイト。

これだ。

---

「ウェブ」をアルファベットで表記するときの表記、ひとによってまちまちだなと思っていて、かなりどうでもいいかもしれませんが気になってしまう。ので、いち意見を表明しておこうと思います。

[World Wide Webに関する初期の文書](http://w3.org/WWW)を見てみると、全体的に「Web」と表記されていました。

現在のW3Cが公開している表記ルール[^1]でも、*単体での利用*については常に大文字で表記するようにと書かれています。

| 表記                                | ルール                                                                      |
| :---------------------------------- | :-------------------------------------------------------------------------- |
| Web（単独で使うとき）               | 常に大文字にする。                                                          |
| Web（フレーズの一部として使うとき） | 「Web」は大文字でも小文字でもよい。                                         |
| Web page / web page                 | 二語で書く。「Web」は大文字でも小文字でもよい。                             |
| Web site / web site / website       | 二語（「Web」は大文字でも小文字でもよい）、または一語（小文字で website）。 |

[^1]: Web (on its own) always capitalize - [W3C Manual of Style](<https://w3c.github.io/manual-of-style/#:~:text=lowercase-,Web%20(on%20its%20own),-always%20capitalize>)

同じくW3CであるWAIのスタイルガイドを見てみましょう。おや…逆のことが書かれていますね[^2]。

| 表記 | 用途                                                               |
| :--- | :----------------------------------------------------------------- |
| web  | 単独で使うときは小文字。形容詞的な用法や、名詞的な用法で使用する。 |
| Web  | 固有名詞やタイトルの中で使用する。                                 |

[^2]: web - lowercase when used by itself, including as an adjective (such as "web accessibility", "web page", "web application") and a noun ("making the web accessible") [WAI Style Guide - W3C Wiki](https://www.w3.org/wiki/WAI_Style_Guide)

「W3CがWebって書いてるからWeb！」という記事を書こうと資料を漁っていた筆者の出鼻は木っ端微塵です。W3Cの中でも意見が割れている模様。

とはいえ、筆者の結論は変わりません。

歴史的にもWorld Wide Webの略称として**Web**という表記が用いられてきたことから、個人的には**Web**を基本表記として採用していきたいと思います。

とくに懸念しているのは、日本語文章の中で同じ意味の英単語を大文字小文字で使い分けてしまうと、結局表記揺れのように受け取られてしまう可能性がある[^3]という点です。すでに筆者はもやもやしてしまっていますし🍣

[^3]: 具体的な調査をされていて、同じ意見の方がいらっしゃいました。参考：[Web/WEB/web どの表記が正しいの？](https://note.com/gita_mw/n/n8db22d918b93)

なんだかもう日本語なら、いっそのこと「ウェブ」のほうが素直な気もしますが、みなさんは「うぇぶ」をどう表記しますか？

ウェブサイト、webサイト、Webサイト。

webブラウザ、Webブラウザ、ウェブブラウザ。

Webアクセシビリティ、ウェブアクセシビリティ、webアクセシビリティ。

…やっぱり個人的にはWebが良いと思うんですけれど、みなさんはいかがでしょうか。
