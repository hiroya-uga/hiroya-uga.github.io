# Writing Guideline

## 技術用語の表記

名前だけでは種別が判断できないため、種別を添えて表記する。

| 種別             | 表記例                                         |
| ---------------- | ---------------------------------------------- |
| HTML要素         | `cite` 要素、`input` 要素                      |
| HTML属性         | `cite` 属性、`type` 属性                       |
| CSSプロパティ    | `color` プロパティ、`grid-template` プロパティ |
| CSS関数          | `rgb()` 関数、`repeat()` 関数                  |
| JS関数・メソッド | `fetch()` 関数、`addEventListener()` メソッド  |
| JS変数・定数     | `count` 変数、`MAX` 定数                       |

## 数字の表記

数えられる・変動する数量は算用数字、慣用句・固有名詞・熟語は漢数字を使う。

| 種別                         | 表記例                               |
| ---------------------------- | ------------------------------------ |
| 数量・回数・序数（算用）     | 2本指、3回、5個、第1引数             |
| 慣用句・成句・熟語（漢数字） | 二人三脚、四捨五入、一石二鳥、二の足 |
